#简要介绍
《雷电大战》是一框射击类游戏，代码全部是本人原创，资源图片部分原创，部分网上搜集。游戏的具体代码实现过程以及游戏效果请看笔者博客地址：[http://blog.csdn.net/evankaka](http://blog.csdn.net/evankaka "http://blog.csdn.net/evankaka")
